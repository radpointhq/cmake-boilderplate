# cmake-boilderplate
Simple CMake setup with one exec and one library and installer (Mac)

For now it is only for Mac
Someday Windows with NSIS installer will be added.

## How to use it?

```sh
cd build
#intialie it
cmake ..
#compile it
make
#package it
make package
```
